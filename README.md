
# Digital Payments Analytics Dashboard (UPI - 2024)

## Project Overview

This project analyzes 250,000+ UPI transactions using Python, Pandas, and Power BI to generate business insights related to digital payment trends, fraud detection, merchant performance, banking performance, and customer behavior.

---

## Tools Used

- Python
- Pandas
- NumPy
- Power BI
- DAX
- GitHub

---

## Key KPIs

- Total Transactions
- Total Revenue
- Average Transaction Amount
- Success Rate
- Failure Rate
- Fraud Rate

---

## Dashboard Features

- Revenue by Sender Bank
- Revenue by Sender State
- Merchant Category Analysis
- Fraud Analysis
- Network Type Analysis
- Device Type Analysis
- Age Group Analysis
- Transaction Status Distribution
- Weekend vs Weekday Analysis
- Interactive Slicers

---

## Business Insights

- Identified top-performing sender banks by revenue.
- Compared transaction success and failure rates.
- Analyzed fraud trends across banks.
- Evaluated merchant category performance.
- Examined customer transaction behavior across age groups and states.
